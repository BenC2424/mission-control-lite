# Inbox Operating Policy (Committed 2026-03-08)

- Inbox is triage-only and owned by `ops`.
- Any inbox task older than 24h requires triage classification and explicit action.
- Safe archival categories: `test/canary`, `duplicate`, `obsolete`.
- `legitimate work item` must move `inbox -> assigned` (owner remains `ops` until dispatch flow).
- `unclear` items remain in inbox with `triage_review_required` note.
- Inbox hygiene is run in telemetry-first mode before any auto-archive enforcement.
